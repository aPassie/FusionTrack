## Quick Start

### Prerequisites
- Node.js installed
- MongoDB Atlas account
- Google OAuth credentials

### Installation

1. Clone the repo
```bash
git clone https://github.com/aPassie/FusionTrack.git
cd FusionTrack
```

2. Install dependencies
```bash
# Backend
cd backend
npm install

# Client
cd ../client
npm install
```

3. Set up environment variables (see `.env` files in backend and client folders)

4. Run the app
```bash
# Backend (from backend folder)
npm run dev

# Client (from client folder)
npm run dev
```
